## angular 是什么?

## 2. 指令

- ng-app 指令  
    `ng-app 指令启动一个AngularJS应用。它定义根元素。它会自动初始化或启动加载包含AngularJS应用程序的Web页面的应用程序。`
- ng-init 指令  
    `ng-init 指令初始化应用程序数据。`
- ng-model 指令  
    `ng-model 指令把元素值（比如输入域的值）绑定到应用程序。`
- ng-bind 指令
    `ng-bind 指令绑定一个变量`
